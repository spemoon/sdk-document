地区接口

  • 请求地址：GET /countries/86/provinces
  • 请求地址：GET /countries/86/provinces/\/cities
  • 请求地址：GET /countries/86/cities/\/districts
  • 是否认证：是

  • 成功返回：

      [
     {
        "id":"id",
        "name":"名称",
        "py":"拼音"
     },
     ...
  ]

上传图片

  • 请求地址：POST /photos
  • 是否认证：是
  • 请求格式：multipart/form-data

  • 请求字段

    • usage_type 图片用途，详细查看 \
    • file 文件

  • 成功响应：

      {
     "src":"图片src",
     "filename":"文件名",
     "url":"图片url"
  }

图片地址格式

  • 原图地址：不公开访问
  • 缩略图地址：/photos/\/\
  • 缩略图实际地址：http://lace-images-thumb.oss-cn-qingdao.aliyuncs.com/\/\
  • 说明
    • photo_src 如：700000000001_5asyj2
    • thumbnail 支持：c75x75.jpg, w1024.jpg
      • c 开头表示裁剪；75x75 表示裁剪的尺寸
      • w 开头表示限定宽；紧跟的数字表示宽
      • .jpg 是图片格式
