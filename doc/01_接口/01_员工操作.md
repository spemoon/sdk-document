### 创建员工
- 请求地址：**POST /organizations/\<int:organization_id\>/workers**
- 是否认证：是
- 请求参数：

        {
           "invite_email": "邮箱账号（必须）",
           "contact": 
           {
              "full_name":"姓名",
              "tels":
              [
                 {
                    "label":"手机号",
                    "value":"手机号内容"
                 }
              ]
           },
           "status": "状态：1在职，2离职，3停职",
           "remark":"备注",
           "employee_num": "工号（预留）",
           "permissions":[ ] //1编辑，2审核员 <ENUM PERMISSION_MASK>
        }

- 请求参数：

        {

           "contact": 
           {
              "full_name":"姓名",
              "tels":
              [
                 {
                    "id":"电话id", //有id表示修改，没有id表示新增

           "remark":"备注",
           "employee_num": "工号（预留）",
           "permissions":[ ] //1编辑，2审核员 <ENUM PERMISSION_MASK>
        }
